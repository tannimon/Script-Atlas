How this works, and what's worth deciding:

Fonts hosted in this repo (e.g. under a /fonts folder) can be loaded by their raw.githubusercontent.com URL using the browser's FontFace API — no backend needed.
The presets above point at real Google Fonts files as a stand-in. Once Adam has specific font files in mind, we'd add them to the repo and swap these preset URLs for our own.
